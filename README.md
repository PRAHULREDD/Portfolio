# P. Rahul Reddy — Edge AI & Computer Vision Portfolio

This is my personal portfolio website, focused on showcasing my work in Edge AI and Computer Vision. It is built to be fast, interactive, and to demonstrate real-world ML technical metrics.

[View Live Portfolio](https://prahulredd.github.io/Portfolio/)

## Tech Stack
*   **Framework**: React 19 + Vite 6
*   **Styling**: Tailwind CSS v4 (Glassmorphism & animations)
*   **Animation**: Motion (for transitions & physics engine)
*   **Icons**: Lucide React
*   **Analytics**: Google Analytics 4 (GA4)
*   **Forms**: Web3Forms

## Interactive Features
*   **Draggable Skills Network**: A physics-based grid where you can interact with and toss skill nodes around the screen using Framer Motion constraints.
*   **ETL Pipeline Visualizer**: An animated CSS graphic in the Experience section showing the "Extract → Process → Train" data flow for the RealMeds project.
*   **Accuracy Migration Chart**: A dynamic bar chart in the Pipra Solutions section that visually compares Haar Cascades vs. MTCNN+FaceNet accuracy gains.
*   **Resume Modal**: An OS-style glassmorphic window for viewing and downloading my resume directly in the browser.

## Getting Started

### 1. Prerequisites
You'll need Node.js and npm installed on your machine.

### 2. Setup
Clone the repository and install the dependencies:
```bash
git clone https://github.com/PRAHULREDD/Portfolio.git
cd Portfolio
npm install
```

### 3. Environment Variables
Create a `.env` file in the root directory (refer to `.env.example`):
```env
VITE_GA_ID=your-google-analytics-id
VITE_WEB3FORMS_KEY=your-web3forms-access-key
```

### 4. Development
To start the local development server:
```bash
npm run dev
```

### 5. Deployment
To push the site to GitHub Pages:
```bash
npm run deploy
```
